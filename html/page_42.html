Listening to view events

You can attach DOM event listeners to the DOM elements using the events

property of view. These events can only be registered on the view element

along with its child elements:

var UserView = Backbone.View.extend({

html: '<button id="btn">Click me</button>',

events: {

'click #btn': 'onButtonClick'

},

render: function () {

this.$el.html(this.html);

return this;

},

onButtonClick: function () {

console.log('Button clicked');

}

});

We added a click event on the button and defined the handler to be called once the

user clicks on that button.

Backbone delegates all of the view events, so that the events are attached even if the

element is not rendered in the DOM. So, if you add one event hash for an element

that is not yet available inside the view DOM, the event will be attached to it as soon

as the element is rendered.
