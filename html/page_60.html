This is an example of a simple ItemView class, where we pass the model and use its

methods to display the data. Have a look, we didn't provide any render() method

definition here. It is because ItemView provides a simple rendering functionality by

default. ItemView has a serializeData() method that sends the model data or the

collection data attached to this view, and the render() method applies this data to

its template and populates the view automatically with generated HTML content.
