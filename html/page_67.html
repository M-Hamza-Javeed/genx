There is a list of users, and when you click on a user item, the user's details are

displayed on the right-hand side of the layout. You will find the complete code

example with all HTML, CSS, and other files in our sample code. Here we will

describe the parts that are critical.

We create a user model and collection first as shown in the following code snippet:

// Change template delimiter to Mustache type

_.templateSettings = {

interpolate: /\{\{(.+?)\}\}/g

};

// User Model

var User = Backbone.Model.extend({

defaults: {

avatar: '',

name: '',

email: '',

phone: '',

twitter: ''

}

});

// Users collection

var Users = Backbone.Collection.extend({

model: User

});
